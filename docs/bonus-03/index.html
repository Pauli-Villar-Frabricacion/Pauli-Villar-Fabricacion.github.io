Bonus 03

Maceta Biodegradable

Intención de Diseño

La idea de este macetero es lograrlo a partir de desechos que existan en mi casa para que replicar mas maceteros sea factible, para crear un huerto, por ejemplo.

Mis expectativas frente a este desafio son bastante positivas, en relacion a los materiales elegidos, creo que cada uno cumple una función especifica dada sus caracteristicas que logrará un resultado óptimo en términos de estructura y estética.

Investigación de materiales

YERBA MATE

La yerba mate usada durante 2 semanas aprox me dio una cantidad considerable como para elegir este material como mi material base para la elaboración de este macetero.

CASCARA DE PAPA

La papa se conforma mayormente de almidón,y a simple vista pensé que el remojo de estas cascaras permitiría que la papa se hidratara soltando el almidón que se conoce como un liquido pegajoso que sirve como pegamento. Por lo tanto la papa seria mi elemento contenedor de este macetero.

PAPEL DE DIARIO Y ALGODÓN

El diario y pétalos que ya fueron ocupados se utilizó ya que al contener celulosa, podría ayudar a darle una textura de tipo engrudo a esta pre-forma de macetero. Cortado en trozos el diario y los pétalos demenuzados para facilitar la hidratación y el triturado posterior

HOJAS SECAS

Las hojas secas fueron parte del material “verde” de este macetero, para darle mas características naturales relacionadas con la fertilidad posterior que el macetero debe tener a modo de ser un buen contenedor para una planta.

Proceso

Paso 1

Disponer de todos los elementos en un recipiente para luego hidratar y dejar reposar al menos un dia.

Paso 2

Luego de que el engrudo pasara dos dias en remojo con abundante agua, se traspaso todo el contenido del recipiente a una juguera donde se trituro hasta lograr un material cada vez mas homogéneo.

Paso 3

Una vez triturado todo, se eliminó exceso de agua y se traspasó el material a un bowl de metal donde se le dio forma con fondo con un bowl mas pequeño situado en su interior. Este se presionó contra el material, contenido en el bowl mas grande y con las manos se le dió forma a las paredes del bowl.

Paso 4

El secado de este macetero fue a la interperie durante un dia, y luego puesto unas horas cercano a una bosca encendida para simular el proceso de horneado del material para obtener un secado mas rapido. Luego se desmoldó y quedo listo para ser usado.

CONCLUSIONES

Si bien se logró el objetivo de configurar este macetero a partir de desechos orgánicos, hubieron varios puntos debiles en el resultado:

1.- Humedad

La razón por la que el proceso de secado fue tan extenso fue debido al exceso de agua de la mezcla, que debio haber sido drenada posterior a triturar los materiales, con un colador o un paño. Esto produjo que la estructura permaneciera muy húmeda, debil y se me hiciera imposible desmoladarla sin causarle daño estructural. Esto demando al menos una semana de secado al aire libre para poder desmoldar

2.- Material “Pegamento”

Independiente de la húmedad, pude observar que la mezcla no desarrollo caracteristicas aglutinantes proveniente de la papa, y esto fué debido a que luego de investigar, me di cuenta que para activar este pegamento natural de la papa, se debe aplicar agua caliente para que libere el almidón, nuestro pegamento.
